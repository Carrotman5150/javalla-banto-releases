# じゃばら番頭 — アップデート配信

**じゃばら番頭**（見積書・請求書・納品書・領収書作成アプリ）のアップデート配信用リポジトリです。

- 最新版は [Releases](https://github.com/Carrotman5150/javalla-banto-releases/releases/latest) からダウンロードできます
- アプリをインストール済みの場合は、起動時に自動でアップデートが通知されます

## インストール方法

1. `jabara-bantou-setup.exe` をダウンロード
2. ダブルクリックしてインストール（Python 不要）
3. デスクトップのアイコンをダブルクリックして起動

## 動作環境

- Windows 10 / 11（64bit）
